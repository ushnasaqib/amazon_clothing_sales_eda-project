**Project Overview**

This project performs an Exploratory Data Analysis (EDA) on Amazon Clothing Sales data to uncover pricing patterns, customer behavior, product ratings, and sales trends. The analysis aims to support data-driven decision-making in e-commerce and retail analytics.

**Dataset Description**

The dataset contains Amazon clothing product sales information including:

- Product prices

- Customer ratings

- Order dates

- Sales-related attributes

**Methodology**

The EDA follows these steps:

- Data Loading

- Data Cleaning

- Handling missing values

- Removing duplicates

- Date formatting

- Exploratory Analysis

- Price distribution

- Rating distribution

- Correlation analysis

- Visualization

- Histograms

- Heatmaps

**Key Findings**

Most clothing items fall within a moderate price range.

Higher-priced products tend to receive better ratings.

Strong correlations exist between price, ratings, and sales volume.
